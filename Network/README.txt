Llopis-Belenguer_et_al_Hostxspecies_abundance.csv
This file containes the host-by-parasite-species abundance matrix, environmental variables and host traits used in the study.

FISH_ID: Host (= fish) codes.
The first number denotes the sampling in a locality (from 1 to 5).
The first letter indicates the Sea (A: Sea of Azov; B: Black Sea; J: Sea of Japan).
The second letter indicates the host species (C: Mugil cephaus; S: Planiliza haematocheilus)
The third or third and fourth letters indicate the locality within the sea
	- Localities in the Sea of Azov (K: Kerch; B: Balaklva; S: Sivash; O: Obytochny; M: Mauripol; U: Utluksky; MO: Molochny)
	- Localities in the Sea of Japan (K: Kiyevka; R: Razdol'naya; P: Poiset; A: Artemovka)
The last two numbers indicate de fish number

Matrix abbreviations:
NA: not available
Month: April (apr); May (may); June (jun); July (jul); August (aug); September (sep); October (oct); November (nov); December (dec)
Host species (Hostsp): Mugil cephaus (Mc); Planiliza haematocheilus (Lh)
Host total length (TL)
Host standard length (SL)
Host height (H)
Host weight (w)
Sex (M: male; F: female: J: juvenile)
